What is HTML?

HTML, Short for HyperText Markup Language, is code that is used to structure content on a web page.

What do you need to make a website?

  • Access to a Computer running either MacOS, Windows, or Linux
  • Visual Studio Code: Download Here
  • A Website Host

The Basic HTML Skeleton:

The DOCTYPE Tag:

The DOCTYPE Tag is a declaration which tells the browser what kind of HTML the code is written in.

The Head Tag:

The Head tag contains metadata about the web page such as its title and its style sheets which we will touch on in a later page.

The Body Tag:

The Body tag contains most of the web pages content. This is where actual data is displayed and most of the coding is done



Other Useful HTML Tags



The h1 Tag:

The h1 tag is used to create a header with large font, often used for titles and other important information. The text "The h1 Tag:" displayed above was actually created using an h1 tag"

The Paragraph Tag:

Often Denoted by just the letter P the paragraph tag is used to display any general text and has some of the most use of any tag in HTML.

The img Tag:

The img tag is used to display images within the content of the web page. An example of this can be seen above with the HTML skeleton example.

The a Tag:

The a Tag is used to create links to other web pages within an HTML document. You've probably already clicked on multiple a tags without knowing it on this web page such as the menu or the link to download Visual Studio Code
