What is CSS?

CSS, Short for Cascading Style Sheets, is a coding language that allows the user to create visually appealing websites by interacting with HTML and HTML tags

For Example:

CSS Can make this text big, bold and purple

How do you add CSS?

Well there are two ways to add them, inline style which is when you add them directly into the HTML tag itself, or external style sheets where you create the elements on a seperate file and link it to the HTML file. We will be primarily focusing on external style sheets.

Lets Take a Look at Classes and Ids

What is a Class?

A CSS Class is an attribute that is used to group HTML elements to apply a specific styling to all of them. Sort of like all of these responses. These are all formatted using classes.

What is an Id?

An Id in CSS is a bit more specific than Classes and is used to identify a single element giving it a unique styling. Typically only used for a single Element.
