What is an API?

API stands for application programming interfact, which lets you communicate with other products or servies without having to directly understand the communication.

Commonly Used APIs

1. Weather API

This API is used to show the weather within your area based on your coordinates. You can see this API on your phone or by even looking up the weather in your web browser.

2. Pay with PayPal

This API is used all the time when shopping on line and is often used in conjunction with other payment options to allow users to pay for services with their paypal accounts.

3. Google Maps

This API does a lot more than you would think at a first glance. This API grants the user full control over geographical coordinates and allows the ability to search a wealth of information on the surrounding area or even the geographical information of the entire earth. You have most likely interacted with this API when using apps like Google Maps or Google Earth.

4. YouTube API

This API allows the user to add the ability to search videos from Youtubes database from any website they add this API to.
