What is JavaScript?

JavaScript is a coding langugage that allows you to create dynamic web pages that can update content, control multimedia, or even animate images. JavaScript works with HTML and CSS to create highly interactive web pages for users.

How Do You Add JavaScript to Your Website?

There are three main ways to add JavaScript into your website. These ways include, Embedding code using the HTML script tag, inline code, and using an external JavaScript file. Lets break down each of these options a bit further.

Embedded JavaScript:

Embedded JavaScript is added directly into the code using the script tag from HTML.

Inline JavaScript:

Inline JavaScript is when you add JavaScript directly to a different HTML tag that isnt a Script tag. For example an a tag.

External File:

External File javascript is when you store all of your JavaScript in an external file and then only call it within your main HTML file. This can keep code organized by sectioning off what code goes where.
